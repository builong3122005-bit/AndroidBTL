# Thư mục model (Các lớp Đối tượng Dữ liệu - Models)

## Các tệp tin trong thư mục:
- **ThongTinDangKy.java**: Đối tượng chứa thông tin đăng ký tài khoản (ID, họ tên, mật khẩu, email, sđt).
- **ThongTinCacQuocGia.java**: Đối tượng chứa mã tiền tệ, tên nước và cờ đại diện của nước đó.
- **ExchangeRateItem.java**: Đối tượng biểu diễn một hàng trong bảng tỷ giá thị trường (mã tiền, tên nước, cờ, tỷ giá so với USD).
- **ExchangeRateResponse.java**: Đối tượng ánh xạ từ JSON trả về của ExchangeRate-API.
- **NewsArticle.java**: Đối tượng chứa thông tin chi tiết một bài viết tin tức.
- **NewsApiResponse.java**: Đối tượng ánh xạ từ JSON trả về của NewsAPI quốc tế.
