# Thư mục api (Giao tiếp mạng - Retrofit Clients & Services)

## Các tệp tin trong thư mục:
- **ApiClient_Price.java**: Khởi tạo và quản lý thực thể Retrofit dùng chung (Singleton) kết nối với ExchangeRate-API.
- **ApiClient_News_en.java**: Khởi tạo và quản lý thực thể Retrofit dùng chung kết nối với NewsAPI.org.
- **ExchangeRateApi.java**: Định nghĩa phương thức gọi API GET tỷ giá tiền tệ.
- **NewsApiService.java**: Định nghĩa phương thức gọi API GET tin tức kinh tế quốc tế.
