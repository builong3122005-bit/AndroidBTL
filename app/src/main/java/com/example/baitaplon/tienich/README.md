# Thư mục tienich (Các tệp Tiện ích - Utilities)

## Các tệp tin trong thư mục:
- **CsvUtils.java**: Các hàm tiện ích phục vụ đọc dữ liệu nến từ tệp tin CSV lịch sử tỷ giá, sắp xếp lại theo thứ tự thời gian tăng dần và lưu thêm bản ghi tỷ giá mới tải về.
- **NguonTinTuc.java**: Các hàm tiện ích quét thông tin (cào dữ liệu) từ trang CafeF, VietNamBiz (bằng Jsoup) và thực hiện gọi API NewsAPI để lấy tin tức kinh tế.
