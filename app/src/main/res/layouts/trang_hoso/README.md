# Thư mục Giao diện trang_hoso

## Các tệp tin layout:
- **activity_profile.xml**: Layout màn hình thông tin cá nhân. Hiển thị thông tin ID tài khoản, Họ tên, Email, Số điện thoại của người dùng hiện hành cùng nút chỉnh sửa thông tin và nút xóa tài khoản.
- **dialog_suahoso.xml**: Layout hộp thoại chỉnh sửa thông tin cá nhân (EditText nhập Họ tên mới, Email mới và Số điện thoại mới).
