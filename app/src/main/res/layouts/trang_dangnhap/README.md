# Thư mục Giao diện trang_dangnhap

## Các tệp tin layout:
- **activity_login.xml**: Layout màn hình đăng nhập. Chứa các ô nhập (EditText) cho ID tài khoản và mật khẩu, nút bấm Đăng nhập, liên kết Đăng ký tài khoản mới và nút bấm trải nghiệm nhanh dưới tư cách Khách.

## 🌗 Hỗ trợ Day/Night Theme:
Màn hình đăng nhập sử dụng màu nền động `@color/background_main` và các nhãn văn bản dùng `@color/text_primary`. Điều này giúp trang đăng nhập tự động thay đổi từ màu sáng sang nền tối huyền ảo tương ứng với cấu hình hệ thống Android.
