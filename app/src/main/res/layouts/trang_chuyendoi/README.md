# Thư mục Giao diện trang_chuyendoi

## Các tệp tin layout:
- **activity_convert.xml**: Layout trang chủ quy đổi tiền tệ. Chứa hai ô hiển thị tiền tệ (trên/dưới), cờ quốc gia, nhãn hiển thị tỷ giá và bàn phím số giả lập tích hợp sẵn các phép tính cộng, trừ, nhân, chia ở nửa dưới màn hình.
- **activity_chon_don_vi_tien_te.xml**: Layout màn hình danh sách lựa chọn tiền tệ. Chứa ô nhập từ khóa tìm kiếm (EditText) và RecyclerView cuộn danh sách quốc gia.
- **item_chon_don_vi_chuyen_doi.xml**: Layout định nghĩa giao diện một dòng trong danh sách chọn tiền tệ (ảnh cờ nước, tên nước và mã tiền tệ).
