# Thư mục Giao diện trang_thitruong

## Các tệp tin layout:
- **activity_thi_truong.xml**: Layout màn hình Bảng giá thị trường. Gồm phần đầu trang hiển thị ngày giờ hiện tại, thanh tìm kiếm và RecyclerView cuộn bảng tỷ giá so với đồng đô la Mỹ (USD).
- **item_exchange_rate.xml**: Layout định nghĩa giao diện một dòng tỷ giá trong danh sách thị trường (hình cờ nước, mã tiền tệ, tên nước và trị giá tỷ giá).
