# Thư mục Giao diện trang_dangky

## Các tệp tin layout:
- **activity_dang_ky.xml**: Layout màn hình đăng ký tài khoản mới. Cung cấp các trường nhập thông tin: ID tài khoản, Họ tên, Mật khẩu, Email và Số điện thoại cùng nút Đăng ký.

## 🌗 Hỗ trợ Day/Night Theme:
Màn hình đăng ký được tối ưu hóa nền và nhãn thông tin qua `@color/background_main` và `@color/text_primary` để tự động đổi màu đen/trắng tương ứng đồng bộ với trang đăng nhập khi bật tắt Dark Mode.
